# Decidim::VotingBudgetsRule

Voting budgets rule.

## Usage

VotingBudgetsRule will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-voting_budgets_rule"
```

And then execute:

```bash
bundle
```
